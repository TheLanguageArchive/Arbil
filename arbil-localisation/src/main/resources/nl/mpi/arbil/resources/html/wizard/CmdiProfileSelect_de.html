Wählen Sie Ihre CMDI-Profile aus

Bitte wählen Sie Ihre CMDI-Profile aus, die Sie verwenden möchten.
Sie können Ihre Auswahl immer noch ändern, indem Sie 'Vorlagen/Profile' im Optionen-Menu benutzen.

Wählen Sie die Profile mit denen Sie beginnen möchten aus der untenstehenden Liste. Wenn dieses das erste Mal ist,
dass Sie den Assistenten verwenden, müssen Sie die Profile erst herunterladen. In diesem Falle kann es einen Moment dauern,
bevor die Liste angezeigt wird.

Wenn Sie auf die Profile zeigen, werden Sie eine Beschreibung dieser sehen.
