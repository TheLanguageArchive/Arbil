Wählen Sie ein Metadaten-Format

Bitte wählen Sie das Metadaten-Format, das Sie verwenden möchten. Beide Metadaten-Formate können
gleichzeitig in Arbil verwendet werden, jedoch verwenden die meisten Nutzer nur eines der unten-stehenden Formate
Indem Sie nur eine Option wählen wird die andere Option aber auch nicht permanent blockiert.

IMDI ist der Isle-Metadaten-Initiative Standard. Mehr Informationen sind
auf www.mpi.nl/IMDI/ zu finden.

CMDI ist die Komponenten-Metadaten Infrastruktur, eine auf Profilen-basierende
Herangehensweise für Metadaten, wobei domänenspezifische Profile designt werden können, die eine Struktur für Metadaten-Einheiten
definieren. Mehr Informationen auf www.clarin.eu/cmdi.
