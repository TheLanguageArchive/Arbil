Selección formato metadatos

Seleccionad el formato de metadatos que queréis utilizar. Ambos formatos se pueden utilizar
simultáneamente en Arbil, pero la mayoría de los usuarios suele utilizar uno de los dos formatos aquí
abajo. Al seleccionar sólo una opción no se deshabilitará de manera permanente la otra.

IMDI significa Isle Meta Data Initiative. Para más informaciones
consultar el sitio web www.mpi.nl/IMDI/.

CMDI significa Component MetaData Infrastructure, una manera especial de utilizar
los metadatos que se sirve de perfiles específicos de un dominio para definir la estructura de una
instancia de metadatos. Para más informaciones, consultar el sitio web www.clarin.eu/cmdi.
