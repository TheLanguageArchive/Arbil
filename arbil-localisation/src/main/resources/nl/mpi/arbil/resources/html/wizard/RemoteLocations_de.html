Entfernte Speicherorte

Im unten-stehenden Textfeld können Sie die entfernten Speicherorte angeben, die im "Entfernten
Korpus"-Baum in Arbil angezeigt werden sollen. Abhängig vom/ von den Metadaten-Format(en), die Sie wählen, ist eine
bestimmte Anzahl von Sammlungen schon vorhanden.

Beachten Sie, dass jede Zeile eine URL zu einer Metadaten-Einheit oder -Sammlung in IMDI oder CMDI Format
sein sollte.
